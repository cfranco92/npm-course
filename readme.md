# COMMANDS

## INIT NPM WITH DEFAULT CONFIG

```terminal
$ npm init -y
```

---

## INSTALL DEPENDENCIES

### Dev

```terminal
$ npm install eslint -D
```

or

```terminal
$ npm install eslint --save-dev
```

### Production

```terminal
$ npm install react
```

or

```terminal
$ npm install react -S
```

or

```terminal
$ npm install react --save
```

### Global (OS)

```terminal
$ npm install -g cowsay
```

---

## LIST PACKAGES

### Local

```terminal
$ npm list
```

### Global

```terminal
$ npm list -g
```
